# Mapbox Demo

Boilerplate Mapbox GL JS avec un fond clair, des points stylises en bulles rouges
avec avatar, des lignes en pointilles, et des popups.

Demo: https://pacman49.github.io/mapbox/

## Contenu
- `index.html` : page principale.
- `style.css` : styles de la page et des popups.
- `script.js` : initialisation Mapbox, chargement GeoJSON, layers et popups.
- `resources/map.geojson` : donnees d'exemple.
- `resources/img/avatar_bulle.png` : image pour les points.

## Lancer en local
- Ouvre `index.html` dans le navigateur, ou
- Utilise l'extension VS Code Live Server.
